## 🚀 Overview

HRMS-Application is a recruitment management system for handling candidates from first contact to final decision. It helps you track resumes, manage interview steps, send email updates, and control who can see each part of the system.

This app is built for Windows users who want a simple way to manage hiring tasks in one place. You can use it to review applicants, move them through a pipeline, and keep hiring records in order.

## 📦 What You Get

- Resume parsing to pull key details from CV files
- Candidate pipeline tracking from new lead to hired
- Interview scheduling tools
- Email notifications for updates and reminders
- Role-based access control for different user types
- REST API structure for stable app use
- FastAPI backend for fast response times
- JWT login support for secure access
- Microsoft SQL Server support for data storage
